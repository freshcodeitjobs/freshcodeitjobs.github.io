Система приема коммунальных платежей
Великобритания
Приложение, которое облегчает интеграцию коммунальных брокеров с крупными поставщиками электроэнергии и газа. Система позволяет эффективно регистрировать конечного пользователя у поставщика коммунальных услуг, а также — управлять счетами и оплачивать их.
Технологии: Vue, Firebase, Quasar, Google Cloud (Firebase, Firestore)
Соцсеть для специалистов в сфере образования
США
Закрытое сообщество для специалистов, которые строят EdTech бизнес. Социальная сеть предназначена для общения пользователей и продвижения информации об образовательных компаниях.
Технологии: Vue, Node.js, MongoDB, Express.js
Платформа для медицинской аналитики
США
Data Science продукт в сфере биоинформатики предоставляет софт для фармацевтических компаний и лабораторий, которые анализируют биомаркеры, необходимые для разработки лекарств и изучения заболеваний.

Это фреймворк, на основе которого ученые:
— запускают другие приложения, написанные на языке R;
— визуализируют данные в форме графиков и таблиц.
Технологии: Clojure, Clojurescript, core.async, reagent (react wraper, hiccup), spec.alpha (writing specs), cljsbuild, figwheel, tinkergraph gremlin, compojue server, honeysql, clojure.test, R code
Платформа для управления Kubernetes
Германия
Проект работает над оптимизацией «облачных» расходов и сохранением предсказуемости затрат. Обеспечивает эффективность и прозрачность любой среды Kubernetes.
Стек: Golang, NodeJS/Go, Postgresql, VueJS, Typescript, Swagger, Material-UI, Storybook
Система логистического менеджмента
Германия
Проект для оцифровки логистики и упрощения мировой торговли. Компания клиента создает масштабируемые и бесперебойные логистические решения, благодаря которым пользователи могут управлять всей цепочкой поставок.

Единый интерфейс для клиентов, который охватывает всю цепочку процессов: от предложения, бронирования, управления документами, отслеживания и до финальной обработки и анализа данных.
Стек: NodeJS, Javascript, Typescript, MongoDB, Kubernetes
